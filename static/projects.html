Check out my Android App!

Elevent is a social media app that allows users to add events that users can see and upvote according to what they like. Users will be able to choose to see what events they want to see and attend based on upvotes, date, or alphabetical order of events.

Adding events is easy. Just click the "+" button and add the event Title, Date, place, and description.

Upvote the event that you like by clicking on the peanut.

Tap the event to see the details of an event.
